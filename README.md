# C-boilerplate

> Starter template for C projects w/ Makefile

## Project Structure
```bash
.
├── bin
│   └── main
├── obj
│   └── main.o
└── src
    └── main.c

```

## Usage

To create the executable file *./bin/main* : 

```bash
make
```

To delete all object files and the executable : 

```bash
make clean
```
